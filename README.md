# # 📊 Smart Anomaly Detection System

A Machine Learning-based web application that predicts industrial process anomalies using **XGBoost** and an interactive **Streamlit** interface.

---

## 🚀 Project Overview

This project builds a supervised ML model to detect abnormal operating conditions in a manufacturing process.

The system:

- Takes process parameters as input
- Predicts anomaly probability
- Classifies risk level (Low / Medium / High)
- Displays decision using a clean, interactive dashboard

---

## 🧠 Machine Learning Model

- Algorithm: **XGBoost Classifier**
- Evaluation Metric: ROC-AUC
- Custom Threshold: 0.70
- Output:
  - Anomaly Probability
  - Final Prediction
  - Risk Level Classification

---

## 🖥️ Application Interface

Built using **Streamlit**, featuring:

- Light modern UI
- Colored risk indicators
- Two-column layout
- Real-time prediction
- Clean dashboard-style result card
